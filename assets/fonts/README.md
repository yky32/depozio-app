# Satoshi Font Setup

## Download Instructions

1. Visit https://www.fontshare.com/fonts/satoshi
2. Click "Download Family" to get all font weights
3. Extract the ZIP file
4. Copy the following font files to this `assets/fonts/` directory:

Required files:
- `Satoshi-Light.otf` (weight: 300)
- `Satoshi-Regular.otf` (weight: 400)
- `Satoshi-Medium.otf` (weight: 500)
- `Satoshi-Bold.otf` (weight: 700)
- `Satoshi-Black.otf` (weight: 900)

Optional (if available):
- `Satoshi-Italic.otf` (style: italic)
- `Satoshi-BoldItalic.otf` (style: italic, weight: 700)

## File Structure

After downloading, your `assets/fonts/` directory should look like:
```
assets/
  └── fonts/
      ├── README.md
  ├── Satoshi-Light.otf
  ├── Satoshi-Regular.otf
  ├── Satoshi-Medium.otf
  ├── Satoshi-Bold.otf
  └── Satoshi-Black.otf
```
